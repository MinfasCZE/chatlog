## Installation:
connect to your mysql in ``databases.cfg``
```sh
"chatlog"
{
    "driver"                        "mysql"
    "host"                          "localhost"
    "database"                      ""
    "user"                          ""
    "pass"                          ""
}
```

connect to your mysql in ``database.php``
```sh
// DB hostname
$servername = "localhost";

// DB username
$username = "";

// DB password
$password = "";

// DB name
$dbname = "";
```
